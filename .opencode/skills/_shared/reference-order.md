# 参照順

AI は原則として次の順で読む。

1. `AGENTS.md`
2. `.owox/project.md`
3. 対象 `.owox/tasks/task-*.md`
4. `docs/project/index.md`
5. `docs/project/glossary/core.md`
6. `docs/project/tech-stack.md`
7. `docs/project/patterns/index.md`
8. `docs/project/architecture.md`
9. 対象カテゴリの `index.md`
10. 対象個票

追加ルール:

- task 系 skill では `docs/project/tech-stack.md` と `docs/project/patterns/index.md` を省略しない
- archive は明示的に必要な場合以外読まない
- glossary の分野別ファイルは必要時だけ読む
- task に `今回読まなくてよい資料` がある場合は従う
- skill は必要になったものだけ読む
