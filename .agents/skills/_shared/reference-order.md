# 参照順

AI は原則として次の順で読む。

1. `AGENTS.md`
2. `.agents/project.md`
3. 対象 `.agents/tasks/task-*.md`
4. `docs/project/README.md`
5. `docs/project/glossary/core.md`
6. `docs/project/architecture.md`
7. 対象カテゴリの `README.md`
8. 対象個票

追加ルール:

- archive は明示的に必要な場合以外読まない
- glossary の分野別ファイルは必要時だけ読む
- task に `今回読まなくてよい資料` がある場合は従う
- skill は必要になったものだけ読む
